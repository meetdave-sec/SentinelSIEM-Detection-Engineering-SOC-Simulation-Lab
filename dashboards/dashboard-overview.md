# SSH Brute Force Dashboard

## Overview

This dashboard provides a visual representation of brute-force attack activity detected in the environment.

It helps analysts quickly identify attackers, targeted users, and attack patterns over time.

---

## Panels

### 1. Top Attacker IPs

Displays IP addresses generating the highest number of failed login attempts.

---

### 2. Targeted User Accounts

Shows which user accounts are being targeted during the attack.

---

### 3. Attack Timeline

Visualizes attack activity over time, highlighting spikes in brute-force attempts.
